# Oppgave til intervju

## Oppsett

Bruk `yarn` for å installere dependency  
Bruk `yarn test` for å kjøre testene  
Bruk `yarn start 'INPUT'` for å teste programmet med regnestykket INPUT

## Oppgave

Du skal lage et program som tar imot et regnestykket i form av en `String` og regner ut korrekt svar. F.eks kjører man `yarn start '((5+4)*(8/2))'` skal man få svaret `36`. Du kan gjøre antagelsen at alle binære oprasjoner vil ha paranteser rundt seg. F.eks `(5+4)` og ikke `5+4`
